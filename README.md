# Shinzo-Abe-murder-Twitter-Sentiment-Analysis
Following the [shocking murder of Japan ex Prime Minister, Shinzo Abe](https://www.bbc.com/news/world-asia-62089486) I wanted to put into practice my skills and know Twitter's perception of this big news.
In order to do that, I needed to scrape the data of Twitter.

## Technologies used
Project was created using:
  * Python3
  * Pandas
  * Numpy
  * Matplotlib
  * TextBlob
  * WordCloud
